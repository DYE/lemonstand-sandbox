Grid help

Adding and deleting rows

New row: ctrl+i or navigate down in the last row.
Delete current row: ctrl+d. Alternatively you can delete a row by hovering it with the mouse and clicking the delete icon.

Navigation

Use arrow and tab (tab or shift+tab) keys to navigate between cells.

Search

The live search feature allows to filter grid records with a search term. Empty the search field to cancel filtering.
